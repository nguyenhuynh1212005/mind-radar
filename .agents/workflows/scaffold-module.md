## Steps
1. Đọc yêu cầu xem cần tạo Module nào (ví dụ: M2, M3).
2. Đọc hợp đồng module tại `docs/moduleContracts.md` và luồng trạng thái tại `docs/ThuatToan.md`.
3. Tạo Implementation Plan liệt kê các file sẽ tạo (Types, State, UI Component, Scoring Adapter).
4. Dừng lại và chờ duyệt (Wait for Approval).
5. Thực thi việc tạo file. Không viết logic chi tiết, chỉ tạo bộ khung.
6. Chạy `npm run typecheck` (nếu đã có cấu hình).
7. Xuất Walkthrough tóm tắt tiến độ.
