# Design System Guard

## Khi nào áp dụng:
Áp dụng cho MỌI task liên quan đến Frontend, UI  components, CSS.

## Hành động bắt buộc:
1. Đọc kỹ file `DESIGN.md` ở thư mục gốc.
2. Áp dụng đúng mã màu (Hex) và Spacing (px).
3. KHÔNG sử dụng màu nền (background-color) nào khác ngoài `canvas` hoặc `surface` trong quá trình test đang diễn ra.
4. BẮT BUỘC sử dụng font Monospace cho bất kỳ thành phần nào hiển thị số liệu thay đổi nhanh (như mili-giây, điểm số).
