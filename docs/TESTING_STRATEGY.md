# Testing Strategy

## Required tests

For every scoring adapter:
- all correct
- all incorrect
- partial correctness
- missing response
- timeout
- focus lost
- suspicious fast response
- changed answer
- export JSON compatibility
- export CSV compatibility

## MVP test targets

M2:
- Stroop interference cost
- Go/No-Go false press rate

M3:
- forward span
- backward span
- manipulation cost

M5:
- Kendall Tau distance
- planning latency
- revision count

M9:
- calibration bias
- calibration error
